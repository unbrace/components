# Modal

## Installation
Add `<div id="modal"></div>` somewhere inside the `<body/>` of the `public/index.html` file.
> This modal component uses React Portal to render it's content into a DOM node. https://reactjs.org/docs/portals.html


## Props

<Props of={Modal} />

## Usage example
<Source code='<Modal
content={ModalExample}
contentProps={{ someProp: someValue }}
size="medium">
{toggleModal => <Button onClick={toggleModal}></Button>}
</Modal>;' />
